Удаление мусорных символов с URL и реализация Search Engine Friendly URLs (ЧПУ) для поиска WordPress.

== Description ==

Удаление мусорных символов с URL и реализация Search Engine Friendly URLs (ЧПУ) для поиска WordPress, разработанный СМИ <a href="https://mobidevices.ru">MobiDevices</a> и компанией <a href="https://1000.tech/ru">1000.tech</a>.

<h3>Две проблемы:</h3>
<ul>
<li>Ссылки <strong>site.com/page</strong> и <strong>site.com/page?</strong> открывают одну страницу, но для поисковиков это две разные страницы с продублированным контентом;</li>
<li>Ссылки поисковых запросов имеют такой формат: <a href="https://mobidevices.ru/?s=Apple">mobidevices.ru/?s=Apple</a>.</li>
</ul>

<h3>Одно решение:</h3>

<strong>MobiDevices Search URL</strong> – SEO-плагин для преобразования URL поиска с формата <a href="https://mobidevices.ru/?s=iPhone">mobidevices.ru/?s=iPhone</a> на <a href="https://mobidevices.ru/search/iPhone">mobidevices.ru/search/iPhone</a>.

Также <strong>MobiDevices Search URL</strong> удаляет мусорные символы в ссылках страниц и записей, что предотвращает появление дублей для поисковиков, вследствие чего ваш контент будет ранжироваться выше.

<h3>Преимущества:</h3>
<ul>
<li>Автоматический 301-редирект со старых страниц на новые;</li>
<li>Устранение дублей в пагинации: при обращении к <a href="https://mobidevices.ru/search/iPhone/page/1">mobidevices.ru/search/iPhone/page/1</a> произойдет редирект на <a href="https://mobidevices.ru/search/iPhone">mobidevices.ru/search/iPhone</a>;</li>
<li>Нет настроек – плагин достаточно включить и он будет выполнять своё предназначение.</li>
</ul>

<h3>Установите наши другие плагины:</h3>
<ul>
<li><a href="https://wordpress.org/plugins/mobidevices-translate-url/">MobiDevices Translate URL</a></li>
<li><a href="https://wordpress.org/plugins/mobidevices-pageviews/">MobiDevices PageViews</a></li>
</ul>
